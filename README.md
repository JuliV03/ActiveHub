# ActiveHub
Plataforma integral para la gestión y reserva de actividades deportivas y recreativas. Proyecto final de Ingeniería en Sistemas de Información (UTN).

ActiveHub es una solución digital diseñada para centralizar la oferta de actividades físicas y recreativas, conectando de manera eficiente a instructores y alumnos. El proyecto nace como respuesta a la dispersión de información y la gestión manual de reservas en el ámbito deportivo.

Funcionalidades Clave
  Alumnos: Búsqueda con filtros avanzados, gestión de agenda personal y sistema de reservas en tiempo real.
  Instructores: Panel de gestión de clases, control de cupos automatizado y visibilidad de inscriptos.
  Administradores: Moderación de contenido y métricas del sistema.

Stack Tecnológico (Sugerido)
  Backend: Java + Spring Boot / Maven.
  Base de Datos: Neo4j / PostgreSQL.
  Frontend: React.js / Angular.
  Gestión: Jira (Metodología Híbrida) y GitFlow.
